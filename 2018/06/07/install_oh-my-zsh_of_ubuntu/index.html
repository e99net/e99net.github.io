一般来说，直接运行

sudo apt-get install zsh

即可。当然也可以下载源Download zsh source，使用curl安装

curl -L https://raw.github.com/robbyrussell/oh-my-zsh/master/tools/install.sh | sh

把zsh设为默认终端

chsh -s $(which zsh)

重启系统，可使用

reboot
